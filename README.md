# VnBug - Vi En Bấc

Tổng hợp bug cho Dev

## Installation

Cần gì thì cài đấy thôi

```bash
pip install vnbug
```

## Usage

```python
import vnbug

vnbug.pluralize('word') # returns 'words'
vnbug.pluralize('goose') # returns 'geese'
vnbug.singularize('phenomena') # returns 'phenomenon'
```
